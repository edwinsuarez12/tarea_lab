ELEMENTOS DE UN SISTEMA DE INFORMACIÓN COMPUTACIONAL

Equipo computacional

hardware necesario para que el sistema de informacion funcione, integrado por computadoras y equipo periferico conectado a ellas.

Recurso humano

son las personas que interactuan con el sistema de información,suministran datos o utilizan el sistema para generar datos.

Fuente

son los datos e información que se introducen en el sistema,es decir,son las entradas requeridas para que el sistema de información funcione

programas

software ejecutado por la computadora, el cual procesa la información de entrada para generar los resultados esperados

Telecomunicaciones

abarca el software y hardware que permite la transmisión en forma electronica de texto e imágenes,entre otros.

politicas y reglas de operación

aplicadas a los procedimientos y mecanismos para hacer que una aplicación de computadora trabaje de acuerdo con estas o regir la parte funcional del proceso administrativo de la empresa.
